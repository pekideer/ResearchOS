# 本机运行环境、OCR 与依赖配置契约

## 1. 适用工具

- `build_local_python_env.ps1`
- `ensure_ocr_needed.py`

## 2. 工具目的

本专题工具用于本机运行环境、Python 依赖、OCR 组件和本地工具链配置。它们服务 ResearchOS 运行，不产生科研结论。`ensure_ocr_needed.py` 同时受 Zotero 父文档契约约束：只有 SQLite 记录显示存在合格的 `needs_ocr` PDF 时，才进入 OCR 依赖检查和处理流程。

## 3. 允许行为

- 创建或检查本机运行环境。
- 定位 OCR 所需组件；只有用户明确批准并传入 `--install` 时才安装缺失组件。
- 输出环境状态、依赖状态和排障建议。
- 读取本机安全范围内的配置文件。

## 4. 禁止行为

- 不把本机私密路径、密钥、代理账号或令牌写入报告。
- 不把 OCR 失败自动解释为文献不可读。
- 不把环境配置写入 Zotero。
- 不在未经用户同意时安装或升级外部依赖。
- `ensure_ocr_needed.py` 默认 `install=false`；无 `--install` 时发现依赖缺失必须停止并给出建议，不得静默安装。

## 5. 验收标准

- 环境路径和配置边界清楚。
- 失败原因可排查。
- 依赖安装或变更经过用户确认。
- OCR 状态只作为文本抽取条件，不作为科研判断。
