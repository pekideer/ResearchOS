# ResearchOS Docs

`docs/` 是 ResearchOS 面向用户和维护者的人读文档入口。

本目录保存说明、治理过程文档、能力映射、审计报告和操作指南。具体科研项目成果不进入本目录；具体项目的读书卡、矩阵、报告、论文材料和审稿回复应写入用户指定项目工作区。

## 目录

- `capabilities/`：能力到项目工作区文件的映射。
- `governance/`：ResearchOS 自身治理计划、过程记录和结构审计。
- `guides/`：使用说明。
- `modes/`：本机科研模式、kit 导出模式等执行模式说明。
- `reports/`：本地系统级人读报告目录。公开仓库只保留目录说明，真实报告默认不提交。

## 边界

- `docs/` 只保存 ResearchOS 框架说明和系统级报告。
- 项目事实源和可复用语料进入本地 `corpus/`；公开仓库只保留说明文件和模板。
- 外部写入执行证据、审批记录和回滚材料进入 `.researchos/outputs/archive/`。
- 不把 `docs/` 当作具体科研项目目录。
