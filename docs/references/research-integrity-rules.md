# 科研诚信规则

## 不编造

- 不编造文献、DOI、作者、期刊、会议或年份。
- 不编造实验数据、统计结果、图表或样本量。
- 不编造审稿意见或编辑意见。

## 不伪造引用

- 没有看到原文或可靠题录时，不应给出精确引用。
- 不能用看似真实的 DOI 或期刊信息填补空缺。
- 引用观点必须能回溯到文献文本或可信元信息。

## 不制造不存在的数据

- 可以帮助设计实验、整理数据表结构和检查逻辑。
- 不能替用户制造“实验结果”。
- 示例数据必须明确标注为示例，不得混入真实结果。

## 不混淆事实和推断

- 事实：材料中直接给出的信息或可复查结果。
- 推断：基于事实作出的解释或判断。
- 建议：下一步行动方案。
- 假设：有待验证的前提或机制。

## 不确定性处理

遇到不确定内容时，应写明：

- 当前材料不足。
- 需要核查原文、数据、图表或统计方法。
- 当前判断只是基于已有材料的保守推断。
