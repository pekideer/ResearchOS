# 中文学术表达规则

## 基本原则

- 先写清楚研究对象、问题、方法、结果和边界，再讨论意义。
- 事实、推断、建议、假设必须区分。
- 不用空泛形容词替代证据。
- 不为追求气势改变技术含义。

## 避免空泛拔高

避免在没有证据支持时使用：

- 显著提升
- 重大突破
- 重要意义
- 填补空白
- 彻底解决
- 极大改善
- 广泛适用

可替代表达：

- 在本文实验条件下，结果显示……
- 与基线方法相比，该方法在……指标上表现更好。
- 该结果提示……可能与……有关，但仍需进一步验证。
- 本研究为……场景下的……问题提供了一种可验证的处理路径。

## Claim 强度控制

- 有统计检验时，可以写“显著差异”，并说明检验方法和阈值。
- 只有趋势图时，写“呈现上升/下降趋势”，不要写“显著提升”。
- 只有单一场景时，不要推广到全部场景。
- 只有相关性时，不要写成因果性。

## 推荐句式

- 本文关注……场景下的……问题。
- 为回答该问题，本文采用……方法，并以……作为评价指标。
- 结果表明，在……条件下，……相较于……表现出……。
- 该结果可能说明……，但其适用性仍受……限制。
